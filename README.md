# 🏟️ MoiGO  - 사람을 잇는 시작 <img src="assets/moigo-logo.png" width=40>
> **"3일 걸릴 모임 준비, 단 3분만에"**  
> 스포츠 관람을 위한 사람과 공간을 연결하는 올인원 플랫폼

혼자서 스포츠 경기를 보는 것이 아쉬우신가요? 함께 응원할 사람을 찾기 어려우신가요?
**MoiGO**는 스포츠 관람 모임 생성부터 장소 예약까지, 모든 과정을 3분 만에 해결하는 플랫폼입니다.

---

## 🎯 문제 정의와 솔루션

### 현재의 Pain Points
| 구분 | 고객 | 가게 사장 |
|------|------|---------|
| **문제점** | • 혼자 보면 재미가 떨어짐<br>• 함께 볼 사람 찾기 어려움<br>• 경기별 가게 정보 부족<br>• 불편한 모임 및 예약 과정 | • SNS 홍보의 한계<br>• 전화 예약의 번거로움<br>• 손님 확보의 어려움<br>• 불분명한 장소 정보 |
| **니즈** | • 함께 응원할 사람<br>• 원하는 경기 중계 가게<br>• 간편한 예약 시스템 | • 효과적인 가게 홍보<br>• 편리한 예약 관리<br>• 안정적인 손님 확보 |

### MoiGO의 해결방안
**올인원(All-in-one) 플랫폼으로 모든 문제 해결**
- 사람 모집부터 장소 찾기, 실시간 예약, 채팅까지 모든 것을 하나의 서비스에서 제공
- 새로운 스포츠 관람 문화 정착: '함께 즐기는 스포츠'라는 가치 실현
- 지역 상권과 상생하는 지속 가능한 비즈니스 모델

## ⚡ 핵심 기능

### 📱 **사용자 기능**

#### 1️⃣ 사용자 인증 시스템
- **일반 사용자 회원가입/로그인**: JWT 토큰 기반 보안 인증
- **사장님 회원가입**: 2단계 인증 (기본정보 → 사업자정보)
- 역할별 맞춤 서비스 제공

#### 2️⃣ 모임 관리 (예약 시스템)
- **모임 생성**: 경기 기반 또는 수동 입력 방식
- **모임 참여**: 기존 모임에 간편 참여
- **실시간 모임 조회**: 원하는 조건으로 필터링
- **내 예약/매칭 조회**: 참여 이력 한눈에 관리

#### 3️⃣ 위치 기반 매장 검색
- **매장 목록 조회**: 다양한 필터링 조건 지원
- **매장 상세 정보**: 편의시설, 스포츠 카테고리 등
- **리뷰 기반 추천**: 신뢰할 수 있는 가게 선택

#### 4️⃣ 안전한 결제 시스템
- **에스크로 결제**: 예약금 자동 관리로 노쇼 방지
- **결제 요청/정산**: 투명한 정산 시스템
- **간편 환불**: 취소 시 자동 환불 처리

#### 5️⃣ 실시간 커뮤니케이션
- **채팅 시스템**: 모임원 및 가게 사장과 실시간 소통
- **채팅방 관리**: 입장/나가기/강퇴 기능
- **메시지 이력**: 모든 대화 내용 보관

#### 6️⃣ 사용자 관리
- **프로필 관리**: 개인정보 및 관심사 설정
- **리뷰 시스템**: 매장 방문 후기 및 평점 등록
- **설정 관리**: 알림, 마케팅 수신 동의 등

#### 7️⃣ 실시간 경기 정보
- **전체 경기 조회**: API 연동으로 최신 경기 일정 제공
- **맞춤 필터링**: 대회, 날짜, 팀명, 경기장 등으로 검색
- **알림 서비스**: 관심 경기 시작 전 푸시 알림

### 🏪 **가게 사장님 전용 기능**

#### 1️⃣ 매장 운영 대시보드
- **실시간 현황 조회**: 오늘의 예약 현황 한눈에 파악
- **예약 관리**: 예약 요청 수락/거절, 예약 목록 관리
- **매출 분석**: 일별/월별 매출 현황 및 통계

#### 2️⃣ 매장 정보 관리
- **기본 정보 수정**: 매장명, 주소, 연락처, 영업시간
- **상세 정보 관리**: 매장 소개, 사진, 특징 등
- **사업자 정보**: 사업자 등록번호, 은행 계좌 정보

#### 3️⃣ 서비스 설정
- **예약 설정**: 예약 가능 시간, 최대 인원 등
- **결제 정보**: 계좌 정보, 정산 주기 설정
- **편의시설 관리**: 제공 서비스 추가/삭제/수정

#### 4️⃣ 스포츠 카테고리 관리
- **카테고리 추가/삭제**: 중계 가능한 스포츠 종목 설정
- **경기 요청 관리**: 고객의 경기 시청 요청 승인/거절

## 🆚 경쟁 서비스 대비 차별점

| 구분 | 일반적인 과정 | **MoiGO** |
|------|------------|-----------|
| **사람 모집** | 에브리타임, 오픈채팅방 등을 돌아다니며 찾아봐야 함 | ⭕ 앱에서 손쉽게 모임 생성/참여 |
| **장소 찾기** | 전화로 일일이 확인하며 원하는 경기를 틀어주는 가게를 찾아야 함 | ⭕ 고객이 직접 원하는 경기를 가게에 요청 |
| **예약 과정** | 전화 혹은 직접 방문을 통해 예약해야 함 | ⭕ 앱 내에서 실시간 예약 요청 및 수락 |
| **커뮤니케이션** | 모임 참여자 및 가게 사장님과의 소통 채널이 제각각이라 혼선 | ⭕ 앱 내 채팅으로 모두와 쌍방향 소통 |

### 4가지 핵심 차별점
1. **올인원 플랫폼** → 번거로운 과정을 하나의 앱으로 통합
2. **예약금 에스크로** → 노쇠 방지 및 안전한 거래
3. **실시간 경기 정보** → API 연동으로 최신 정보 제공
4. **가게 사장 소통** → 직접적인 요청/응답 시스템

## 🎯 시장 진입 전략

### 목표 시장
```
🇰🇷 대한민국 전체 스포츠 관람 시장 (TAM)
    ↓
🏙️ 부산 지역 전체 스포츠 관람 시장 (SAM)  
    ↓
👥 부산 내 MZ세대 스포츠 관람 시장 (SOM)
```

### 마케팅 전략
**1단계: 부산 지역 집중**
- 부산대학교 축제 부스 운영  
- 지역 스포츠바 직접 영업
- 롯데 자이언츠/부산 아이파크 연계 이벤트

**2단계: 전국 확장**
- 타 지역 대학교 진출
- 프랜차이즈 스포츠바 파트너십
- 인플루언서 마케팅
