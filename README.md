## 🎯 시장 진입 전략

### 목표 시장 (TAM → SAM → SOM)

```
🇰🇷 대한민국 전체 스포츠 관람 시장 (TAM)
    ↓
🏙️ 부산 지역 전체 스포츠 관람 시장 (SAM)  
    ↓
👥 부산 내 MZ세대 스포츠 관람 시장 (SOM)
```

### 마케팅 전략

**1단계: 부산 지역 집중**
- 부산대학교 축제 부스 운영
- 지역 스포츠바 직접 영업
- 롯데 자이언츠/부산 아이파크 연계 이벤트

**2단계: 전국 확장**
- 타 지역 대학교 진출
- 프랜차이즈 스포츠바 파트너십
- 인플루언서 마케팅
